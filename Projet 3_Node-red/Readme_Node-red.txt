
Readme projet 3: Node-red

l'objectif de ce projet était d'afficher la température du CPU et/ou de paris
sur un dashboard

le processus est le suivant:
on installe et configure node-red sur notre raspberry
une fois installer, on l'ouvre
on installe le dashboard et la carte des température 
à l'aide des fonctions et des éléments du dashboard, on crée un code permettant de 
d'afficher la temperature du CPU et de Paris
biensur nous pouvons décider d'afficher selon notre convenance (jauge, chart) en modifiant
les couleurs...
une fois récupperer la température, on modifie celui ci pour ne obtenir que la 
valeur de cette dernière, pas de chaine de caractère.

ensuite un insère manuellement un boutton permettant de réactualiser la température
par l'utilisateur
