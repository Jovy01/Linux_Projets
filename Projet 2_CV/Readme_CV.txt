
Readme projet 1: CV

l'objectif de ce projet etait de faire un CV en html et de l'exposer sur un serveur sur la Raspberry pi

la principale tache de ce projet à été de se familiariser avec le langage html pour celà,
les vidéos et tuto sur youtube ont été d'une grande importance.

après avoir compris les bases du html
on télecharge le logiciel sublime text
on créè un fichier .html qu'on l'ouvre par la suite avec le logiciel sublime text

une fois ceci fait, il faut tout simplement mettre en pratique les connaissances acquisses graces aux vidéos
ouvrir et fermer html, l'entete, le titre et ensuite le corps du CV en rajoutant les balises permettant de souligner, 
mettre en gras...

pour ajouter une image, il suffit d'inserer l'image dans le meme dossier et de l'appeller dans notre code
